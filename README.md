# Bronzed & Boujie — website

**Live preview:** https://gazelleryanwalker.github.io/bronzed-and-boujie-preview/

## Files
- `index.html` — the full site (self-contained; logo, photos, palette all embedded)
- `assets/hero.mp4` — hero background video (served as a separate file)
- `assets/` — source images (logo, agate, photos) kept locally for future edits

## Update the live site
```
git add -A && git commit -m "your change" && git push
```
GitHub Pages rebuilds at the same URL in ~1 minute.
